About

This website is a Calculator of the carbon footprint of a website based on this API This website is made for a class and is intended as a learning project. For a professional website that does the same thing visit websitecarbon.com.

Data Collection

When you enter a URL into this website it collects the URL you entered and the date in order to show if this website has been entered before.
